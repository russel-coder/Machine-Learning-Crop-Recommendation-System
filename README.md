# 🌱 Crop Recommendation System

This project predicts the most suitable crop based on soil and environmental conditions using Machine Learning.

## 🚀 Features
- Predicts best crop using soil nutrients and climate data
- Uses Random Forest Classifier
- Provides feature importance analysis
- Includes model evaluation and robustness testing

## 📊 Dataset
Kaggle Crop Recommendation Dataset

## 🧠 Model
- Random Forest Classifier
- Achieved ~0.99 accuracy
- Cross-validation used for stability check

## 📈 Features Used
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Matplotlib
